# 资产库量产 SOP(给队友版)

> 目标:把抖音家装视频变成 App 里可用的 3D 家具资产。
> 流程里只有两个人工环节:**① 收集视频** 和 **③ 审核**,中间全自动。
> 单件资产生成约 40 秒,一小时能产 ~90 件;多人同时投喂自动排队,随时可以加料。

## ① 收集视频(任何人,随时)

把抖音**分享链接**贴进共享文档(一行一条),或者直接把**下载好的视频文件**发到约定位置。

**什么样的视频出活率高**(重要,决定废品率):
- ✅ 家装/家居好物类:房间改造、家具开箱、软装分享
- ✅ 画质 ≥720p,光线好,镜头稳
- ✅ 家具有**完整、清晰、无遮挡**的露出(环绕拍摄最佳)
- ❌ 快剪/闪切、滤镜很重、家具永远只露一角的不要
- ❌ 纯口播、纯效果图/渲染图的不要

一条 30-60 秒的家装视频通常能出 3-8 件资产。

## ② 自动处理(不用管)

运营侧跑一条命令,之后全自动:下载 → 逐帧检测家具 → 跨帧追踪 → 挑最清晰的一帧抠图
→ AI 生成 3D 模型 → 自动打标签(品类/颜色/材质/风格)→ 入库。
每个视频同时生成"时空索引",这就是 App 里"暂停视频圈家具"功能的数据。

```
# (运营侧命令,队友不需要执行)
python tools/batch_produce.py links.txt        # 链接列表
python tools/batch_produce.py ~/videos/        # 本地视频目录
# 中断重跑会自动跳过已完成的,失败的单条跳过不影响批次
```

## ③ 审核(每天 10 分钟,手机可做)

打开审核页(网址群里发),对每件新资产:

| 判断 | 操作 | 标准 |
|---|---|---|
| 模型能看、像原物 | **通过** | 转一圈没有大破洞/飞面,颜色材质大差不差 |
| 模型崩了/不是家具 | **拒绝** | 破洞严重、糊成一团、把窗帘生成了一坨 |
| 和已有资产是同款 | **合并** | 页面顶部"疑似重复"区会自动提示,确认后点保留哪个 |

宁缺毋滥:demo 现场用户看到什么取决于这道闸,拿不准就拒绝。

## 产能与分工建议

- 收集:全员顺手做,目标先攒 **50 条优质视频**(≈300 件资产,远超 demo 需要)
- 审核:一人主责即可(量不大),标准拿不准的丢群里讨论
- 生成排队:GPU 一件一件串行做,白天投喂晚上收货是常态,不用盯

## 常见问题

- **链接下载失败?** 个别视频有防盗链,会在批次汇总里列出来;手动下载成 mp4 再喂一次即可。
- **同一家具在多个视频出现?** 系统会按标签自动提示"疑似重复",审核时合并,越合并库越干净。
- **生成的模型能直接用在 App 里吗?** 能,审核"通过"即上架,App 端实时可见。

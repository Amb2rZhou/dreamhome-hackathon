## 1. 产品转向(新模型)

核心是一个**来自抖音视频的 3D 资产库**:

- 每个资产 = **某个视频 + 某个时间戳 + 某个画面区域**里那件家具**独一无二的样子**。
- **组件库里**:点一个资产 → 跳回它的**原始视频**;可**收藏**;可**组装**进 3D 房间。
- **刷视频(暂停时)**:
  - 已生成的元素 → 画面上**跳动圆环**,点开去库里看/用。
  - 未生成的元素 → 没圆环,可**框选抠图**存入库。

**明确不做「视觉相似度去重」**:用户要的就是那个视频里那件的独特样子,相似款顶替 = 没意义。
"已生成→圆环"是**身份查表**(videoId+时间戳+区域),不是相似度匹配。成本因此天然由"有多少不同元素被真收集"封顶。

## 2. 后端能力清单 + 自建 vs API 决策

判据:量大不大 / 单价 / 专不专门。

| 能力 | 自建 or API | 结论 |
|---|---|---|
| **ASR 语音识别** | ✅ **用云 API** | 讯飞语音听写 WebAPI(免费 500次/天,浏览器可直连)或阿里云智能语音交互(NLS,同账号)。轻、便宜、中文成熟、免 GPU,自建无收益。前端可直接接,后端顶多给个签名小接口防 AppSecret 泄露 |
| **抠图 SAM2** | 🔧 **自建** | 交互选区 + **兼做视频 tracklet(闪烁锚点的数据源)**;追踪这活 API 做不了。抠图是它副业 |
| **图生3D TRELLIS** | 🔧 **自建** | **批量回填量大**,自建单价便宜 5–15×,这是买卡省钱的意义。少量现场可退**腾讯混元3D / Tripo API**兜底(`backend/` 已有 tripo provider) |
| **视频入库检测+追踪**(隐含,别漏) | 🔧 **自建** | GroundingDINO / YOLO-World 检测家具 + SAM2 video 追踪 → 产出 tracklet。是"闪烁"能存在的前提 |

模型权重全部走 **ModelScope 魔搭**(阿里的,大陆内网快、免流量费),别用 HuggingFace CDN(大陆慢/断)。

## 3. 闪烁锚点方案(核心难题:半露 + 运动)

**判断:绝不在客户端实时算,离线预计算。** 资产天然绑定"视频+时间+区域",位置可提前算好存下。

### 入库时(离线,后端,每视频一次)
1. 抽帧 → **镜头切分**(PySceneDetect,避免跨镜头乱追)
2. **检测**家具(GroundingDINO/YOLO-World,prompt=sofa/chair/lamp/table/cabinet/plant)
3. **SAM2 视频预测器**把每个物体 mask 沿全片传播 → **tracklet**:
   ```
   { objectId, label, assetId(初始null),
     frames:[ {t, bbox, 可见mask质心, 可见面积占比, visible} ... ] }
   ```
4. 存库(按 videoId)。有人从某 tracklet 生成 3D → 回填 `assetId` → 该处"available"。

### 播放时(客户端,只查表不算 CV)
- 拉该视频 tracklet JSON(很小);`timeupdate`/暂停时二分当前时间 → 插值 bbox → 在**可见 mask 质心**画跳动圆环。

| 难点 | 解法 |
|---|---|
| 暂停只露一半 | 锚在**可见 mask 质心**(非几何中心);可见面积<阈值(~8%)不标;全遮挡 `visible=false` 圆环淡出 |
| 播放中在动 | 存了**逐帧框**,按渲染帧率在关键帧间插值,圆环跟着走;暂停即停在该帧 |

**收益:SAM2 一鱼两吃**——交互抠图 + 视频 tracklet,少部署一个模型。
**路演:** 对 demo 视频**提前跑完离线管线**,tracklet JSON + 预生成 GLB 打包 → 现场闪烁纯查表,零推理、零外网、断网可演。

## 4. 算力方案

### 结论:租,别买。全阿里云(省心)或阿里云+AutoDL(更省)。

- **一台 gn7i(A10 24G)ECS 就能同时跑 SAM2 + TRELLIS**(ASR 已外包给 API)。
- **卡由 TRELLIS 的 24G 决定**;SAM2 很轻,别为省卡钱降到 T4 让 TRELLIS 又慢又 OOM。
- **学习/搭建/批量回填** → **抢占式(spot)A10 ~¥1–3/时**(设 1 小时保护期 + 释放时保留云盘)。
- **路演当天那几小时** → **按量 A10**(稳定不被回收)。
- **用完必停**:实例→停机→选**「节省停机模式(关机不收费)」**;**千万别包年包月**。停机后只剩云盘 ~¥2–4/天。

### 成本(整个 demo+评奖期)
| 方案 | 花费 |
|---|---|
| 阿里云 spot 为主 | **~¥300–400** |
| 纯阿里云按量 | ~¥800–1,200 |
| 买 4090 整机 | ¥1.5–2 万(要跑 ~90 万次才回本,别买) |

按量 A10 ~¥9.5/时是正常价,只在开机时算钱;"开→用→停"总账才几百块。

### 中国网络适配(路演武汉,评委不开 VPN;开发者当前在墙外)
- 🚫 `webkitSpeechRecognition` 走 Google,大陆不可用 → ASR 必须用国内 API(讯飞/阿里)。
- 🚫 HuggingFace CDN 慢/断 → 权重走 **ModelScope**。
- 🚫 fal.ai / vercel / github.io 默认域大陆不稳 → 前端托 **COS/OSS 默认域**(免 ICP 备案)。
- 端点就近武汉:阿里云**华东(上海/杭州)**,<30ms。
- **路演前务必现场裸网络实测**三个端点 + ModelScope 下载 + 页面加载延迟。
- ⚠️ 账号现实:阿里云/AutoDL 都要**国内实名+境内支付** → **武汉队友持有账号**,开发者 SSH 进去部署。

### 进行中 / 下一步(按顺序)
1. ⬜ 队友注册**阿里云** + 实名 + 充值(¥200);给开发者 RAM 子账号或主账号。
2. ⬜ 买 **gn7i(A10 24G)ECS**:
   - 付费:学习期**抢占式(spot)**,路演当天再开**按量**;
   - 镜像 **Ubuntu 22.04** + **勾「安装 GPU 驱动」CUDA 12.x**(不勾=开机没驱动,最坑);
   - 系统盘 **ESSD 100 GiB**;分配**公网 IP**(按流量);
   - 安全组放行 **22 + 8000**;登录用**密钥对**(`.pem` 存好,只下一次)。
3. ⬜ 用**阿里云网页 Workbench 终端**(实例→远程连接)连入,跑 `nvidia-smi` 确认 A10+CUDA。
   - 沙盒里的 AI 助手**无法帮你 SSH**(出网受限 + 不该拿你私钥操作你的机器);用网页终端粘命令即可,不用碰本地终端/`.pem`。
4. ⬜ 部署 **SAM2 抠图**(FastAPI /segment,ModelScope 拉权重)→ 本地/公网调通。
5. ⬜ 部署 **TRELLIS 图生3D**(异步 job:提交→轮询,对齐 `backend/` 现有 routers/store)。
6. ⬜ **视频入库追踪脚本**(检测→SAM2 tracklet→导出 JSON):把"闪烁"数据链跑通(最不确定也最出彩)。
7. ⬜ **ASR 接入说明**(讯飞 WebAPI 或阿里 NLS)交给前端队友。
8. ⬜ **种子资产回填**(spot 批量跑 TRELLIS)。

### 踩坑备忘
- 阿里云按量**停机要选「节省停机模式」**,否则普通关机仍计费。
- 买实例**必须勾「安装 GPU 驱动」**;没勾则 `nvidia-smi` command not found。
- spot 实例设**保护期 + 释放时保留云盘**,避免被回收丢环境。
- 安全组端口不放行 → 公网调不通(最常见"连不上"原因)。

# 🏠 DreamHome — 把刷到的"好看"，摘抄成你真实的家

> TikTok Hackathon · 赛道四｜视觉搜索：视觉 + 语音，所见即所想

## 灵感来源

我们刷了很多探家博主、装修博主的视频——只是觉得**很好看，但很难保存下来**。点赞、收藏之后，那些美好的软装、配色、家具就沉在收藏夹里，永远变不成"装饰自己的家"的灵感来源。

所以我们想做一本**家的灵感摘抄本**：

- 刷视频时看到喜欢的沙发/吊灯/一面墙 → 暂停、圈选、**摘抄下来**
- 线下逛店、逛朋友家看到心动的家具 → 拍一下、**摘抄下来**
- 脑子里有个说不清的家具 → 随手画两笔、**画出来**

所有摘抄下来的东西都变成**真实的 3D 组件**，最后在自己的户型里组装起来——从"看到别人的好看"，一步步拼成"自己梦想中的家"。

## 核心体验闭环

```
看见（视频圈选 / 线下拍照 / 手绘）
   ↓ 摘抄
灵感素材库（每个元素都是带信息的 3D 组件）
   ↓ 组装
扫描户型图 → 生成自己家的 3D 空间 → 把摘抄的元素摆进去
   ↓ 分享
邀请抖音好友来"梦想中的家"里玩 🌐
```

最后一步是我们觉得最有想象力的部分：**梦想之家本身成为一个可被访问的社交空间**——很元宇宙，但每一件东西都来自你真实生活里"看见并心动"的瞬间，未来还可以一键导向真实购买，把梦想之家变成真实的家。

## 四个"摘抄"入口

| 入口 | 场景 | 交互 |
|---|---|---|
| 📱 视频圈选 | 刷探家/装修视频 | 暂停 → 圈选元素 → 识别 + 语音追问 → 存入素材库 |
| 📷 线下扫描 | 逛家具店、朋友家 | 拍照（1-3 张）→ 生成 3D 组件 |
| 📐 户型图扫描 | 开始装扮自己的家 | 拍/上传户型图 → 自动生成 3D 空间 |
| ✏️ 边画边生成 | 想要的东西世上还没有 | 画板手绘 → 实时生成 3D 家具 |

## 与赛道四的契合

- **视觉**：拍照识别世界（线下扫描、户型图），在抖音短视频中暂停、圈选、继续探索（视频摘抄）
- **语音**：圈选后语音追问（"这是什么风格？配我家吗？"）、语音操控组装（"把沙发挪到窗边"）、语音修正尺寸（"这个柜子是一米二的"）
- **"一次看见，自然延展为理解与互动"**：看见 → 摘抄 → 组装 → 邀请好友进入，一条完整的体验重组链路

## 技术方案

详见 [docs/tech-feasibility.md](docs/tech-feasibility.md)。

一句话版本：**一个 image-to-3D 生成底座，吃下全部三种摘抄输入（视频截帧 / 实拍照片 / 手绘草图），配合户型图 → 3D 空间生成，在 Web 3D 编辑器里完成组装，语音贯穿全程。**

## Demo 优先级（按演示脚本倒推）

1. **核心（70% 精力）**：视频圈选摘抄 → 素材库 → 3D 组装编辑器 + 语音操控
2. **户型图 → 3D 空间**：多模态大模型解析户型图，three.js 拉起房间壳
3. **线下拍照 / 手绘生成**：复用同一 3D 底座，低成本白送的两个 feature
4. **加分项**：`<model-viewer>` AR 预览、好友访问梦想之家

## 已知风险与对策

- **3D 生成延迟（30–120s）**：演示物料预生成 + 缓存
- **生成模型无真实尺度**：品类先验（沙发默认 2.2m）+ 语音修正
- **视频截帧质量参差**：SAM 跨帧追踪取最清晰帧 / 图像增强兜底
